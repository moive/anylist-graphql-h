## Description

The project is nest with graphql

## Project setup

```bash
# clone project
$ git@github.com:moive/anylist-graphql-h.git
```

```bash
$ npm install
```

```bash
# Run docker and create container
$ docker compose up -d
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
